# Conhecendo as estruturas de dados

## Conceito

"É um modelo que usamos para organizar, armazenar e manipular informações dentro de um programa de forma eficiente."
  > Dados do programa - "Alimentos"

  > Memória do computador - "Despensa"

## Estrutura de dados lineares

Numa estrutura linear, os elementos são organizados em uma sequência, um após o outro, formando uma linha lógica.

**Principais estruturas**
- Vetores
- Listas
- Pilhas
- Filas

## Onde usar na prática?

- **LISTA** – gerenciar os contatos.
- **FILA** – gerenciar os pedidos que chegam em um site de e-commerce.
- **PILHA** – implementar a função “desfazer” (Ctrl+Z) em um editor de texto.
